Colors in CSS
Name RGB (Red Green Blue) HEX (Hexadecimal) HSL (Hue Saturation Lightness)
Setup Three values from 0 to 255 Six values from 0 to F One value from 0 to 360 and two values from 0% to 100%
Example Red: rgb(255, 0, 0) Green: #008000 Blue: hsl(240, 100%, 50%)
Extra One can add the value a (from 0/0% to 1/100%) to all three of the above, which defines how transparent the color is with 1 being fully visible and 0 completely transparent
